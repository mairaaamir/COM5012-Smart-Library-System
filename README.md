# Smart Library Management System (SLMS)

## Module
COM5012 Object Oriented Programming

## Assignment
Coursework 1 – Smart Library Management System

## Description
This project implements a partial Smart Library Management System using Object-Oriented Programming in Python.

The system demonstrates:

- Encapsulation
- Inheritance
- Polymorphism

## Features
- Search for books
- Borrow books
- Return books
- Book status tracking
